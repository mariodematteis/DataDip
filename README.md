# DataDip (DEPRECATED)

⚠️ **This project is deprecated and no longer maintained** ⚠️

## Project History

DataDip was developed between June and July 2021. It is no longer actively maintained or updated.

## Project Overview

DataDip was a Java-based application for financial data analysis and trading simulation. It was developed using JavaFX for the user interface and included features for connecting to financial data sources, setting trading conditions, and running simulations.

## Deprecation Notice

This project is no longer actively maintained or supported. It was last updated in its current form and is not recommended for use in production environments.

## Historical Features

- User interface for setting up trading simulations
- Connection to external financial data sources
- Ability to set custom trading conditions
- Support for various financial instruments
- Basic charting capabilities

## Technology Stack

- Java
- JavaFX
- JSON for data handling

## Why This Project is Deprecated

1. Outdated dependencies and libraries
2. Lack of modern security practices
3. Limited scalability
4. Better alternatives now available in the market

## Alternatives

For those interested in financial trading applications, we recommend exploring more up-to-date and actively maintained projects. Some popular open-source alternatives include:

- [Quantconnect Lean](https://github.com/QuantConnect/Lean)
- [TA-Lib](https://github.com/mrjbq7/ta-lib)
- [Zipline](https://github.com/quantopian/zipline)

## Historical Documentation

While this project is no longer maintained, you can find the original structure and main components in the following files:

- `Main.java`: The entry point of the application and main logic
- `Controller.java`: The controller for the JavaFX UI
- `Conditions.java`: A class representing trading conditions
- `sample.fxml`: The FXML file defining the user interface layout
